# üíñ Romantic Valentine App

A premium, interactive Valentine's Day web application built with **React**, **Vite**, and **TypeScript**. This app features a playful "Yes/No" button logic, stunning glassmorphism design, and magical ambient heart animations.

## ‚ú® Features

- **Premium Visuals**: High-end "Glassmorphism" UI with pearl-pink gradients and soft breathing animations.
- **Interactive Background**: Magical heart trails that follow the cursor on desktop and burst on touch in mobile.
- **Ambient Atmosphere**: A rich field of 70 floating pink hearts that fill the screen immediately.
- **Playful Logic**: 
  - The "Yes" button grows aggressively with every "No" click.
  - The "No" button cycles through 20 custom messages (Sinhala & English).
  - Smart layout prevents button overlap during scaling.
- **Fully Responsive**: Optimized for mobile (320px+), tablets, and desktop screens.
- **Personalized Celebration**: A beautiful celebration page with custom romantic messages and animations.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  **Clone or Download** the project files.
2.  Open a terminal in the project directory.
3.  Install dependencies:
    ```bash
    npm install
    ```

### Execution

- **Development Mode**:
  ```bash
  npm run dev
  ```
- **Build for Production**:
  ```bash
  npm run build
  ```
- **Preview Production Build**:
  ```bash
  npm run preview
  ```
